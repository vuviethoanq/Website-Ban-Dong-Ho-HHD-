# 🛒 E-Commerce Website

## 📌 Giới thiệu
Website bán hàng với giao diện thân thiện, hỗ trợ quản lý sản phẩm và người dùng.

## 🚀 Công nghệ
- HTML, CSS, JavaScript
- ASP.NET Web Forms (C#)

## ⚙️ Chức năng
- Hiển thị danh sách sản phẩm
- Thêm / sửa / xóa sản phẩm
- Đăng nhập / đăng ký
- Quản lý đơn hàng

## 👨‍💻 Vai trò
- Thiết kế UI
- Xây dựng logic backend
